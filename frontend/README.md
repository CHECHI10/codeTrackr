# CodeTrackr Frontend

React + Vite frontend for CodeTrackr.

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run preview
```

## Environment

Create `.env` from `.env.example`:

```bash
VITE_API_URL=http://localhost:3000/api
```

For Vercel, set `VITE_API_URL` to the deployed Render backend URL ending in `/api`.
